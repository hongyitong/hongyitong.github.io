溜须拍马

发表于 2017-07-15 更新于 2024-04-09 分类于 管理相关

关于升职，现在的老板只要是根据个人的偏好，而不是有固定的、明确的考核指标，所以，导致属下往往根据老板的喜好、迎合老板的喜好去做事情。最终变成老板身边都是一些溜须拍马的人。而老板关注的都是宏观的、表面上的东西，导致大家都是做所谓的大事、重要的事情，而没有人去踏踏实实做好基础性的工作，因为这些工作是不出彩的、是老板看不见的。而这些基础性的工作往往又是决定公司的关键性因素。
而职场和人生中所谓的做最有价值的事情，而这些事情就是能出彩的，能让老板看见的，能让你升职的。

信息化顶层设计

发表于 2017-07-14 更新于 2017-07-18 分类于 技术相关

信息化顶层设计，或者说信息化规划，对企业的信息化具有关键性的作用。 做好顶层设计，才能在保证目标明确的情况下按部就班的开展信息化工作，再困难都能达到目的，就像习主席说的：“只要路走对了，就不怕遥远。”。 在做信息化规划的时候，往往会忽视了数据架构的设计，而这个却是最容易导致问题的地方。俗话说，三分技术、七分管理，十二分数据。数据是信息化的血脉，如果数据架构有问题，企业的血流不顺，信息化反而会成为企业的阻力。 很多企业没有重视信息化顶层设计，不重视数据架构的设计，信息化的工作就像踩西瓜皮一样，滑到哪算哪。信息化的工作让业务牵着鼻子走，换一个领导，换一套信息化工作。 信息化的顶层设计最好在企业信息化刚开始的时候就思考，否则在有一定的积累之后，存在很多异构系统、存在很多维度不一致的数据的情况下，再进行信息化再造工作，难度会大很多，就像俗话说的船大难掉头。

如何用牛顿法求一个数的平方根

发表于 2017-07-10 更新于 2024-04-01 分类于 文史数理

对x的平方根的值一个猜想y。
通过执行一个简单的操作去得到一个更好的猜测：只需要求出y和x/y的平均值（它更接近实际的平方根值）。
例如，可以用这样方式去计算2的平方根。

猜想 商 平均值
1 2/1=2 (2+1)/2 = 1.5
1.5 2/1.5=1.3333 (1.3333+1.5)/2 = 1.4167
1.4167 2/1.4167=1.4118 (1.4167+1.4118)/2=1.4142
1.4142 ... ...
继续这一计算过程，我们就能得到对2的平方根的越来越好的近似值。

IC、RFID、NFC关系与区别

发表于 2017-04-17 更新于 2017-07-12 分类于 技术相关
  • IC卡 (Integrated Circuit Card，集成电路卡) 有些国家和地区也称智能卡(smart card)、智慧卡(intelligent card)、微电路卡(microcircuit card)或微芯片卡等。它是将一个微电子芯片嵌入符合ISO 7816标准的卡基中，做成卡片形式。所以非接触式IC卡又被称为射频卡或者电子标签。IC卡是指集成电路卡，一般用的公交车卡就是IC卡的一种，也应用在门禁管理、身份证明和电子钱包。IC卡有别于磁卡，所以说IC卡消磁纯属伪科学！简单来说，其工作原理就是读卡器发射一个根据信息变化的电磁波。卡片内部的感应线圈把这个电磁波转换成感应电流，用以传递信息和驱动芯片工作。
