虚拟机密度快速计算方法

发表于 2020-03-15 分类于 技术相关

https://forum.huawei.com/enterprise/zh/thread-260441.html # 1.1.1 虚拟机密度快速计算 华为FusionCloud云计算vCPU资源计算公式（MHz）： vCPU 资源 = 物理CPU个数 * 物理CPU核数 * 单核线程数 * CPU频率 举例：1个CPU，双核，每核2个线程，3.0GHz，那么vCPU资源 = 1 * 2 * 2 * 3.0GHz = 12GHz = 12000MHz。 FusionCompute发放虚拟机流程中可对CPU资源进行限制，有2个重要参数： 份额预留：分配的vCPU资源最小值 份额限制：分配的vCPU资源最大值 份额预留<=份额限制 只有将份额预留选最大值时，vCPU独占1个物理CPU线程 举例：创建1台2U2G的虚拟机，份额预留最大值为 2 * 3000MHz = 6000MHz。 虚拟化场景，重载，许多情况下1台虚拟机每vCPU独占1个物理CPU线程。因此，为了云计算工程师快速方便估算物理服务器可承载的虚拟机数量，可认为1个物理CPU线程 = 1个vCPU。 举例：虚拟化场景，10台服务器，每台服务器2路8核双线程2.8GHz的CPU，提供的vCPU个数 = 10 * 2 * 8 * 2 = 320个vCPU，可承载2U2G的虚拟机160个。 桌面云场景，重载，许多情况下每个物理CPU核可承载4台虚拟机，因此，为了云计算工程师快速方便估算物理服务器的虚拟机密度，可认为 1个物理核 = 4个虚拟机。 举例：桌面云场景，10台服务器，每台服务器2路8核双线程2.8GHz的CPU，虚拟机密度 = 10 * 2 * 8 * 4 = 640 个虚拟机。

总结：

虚拟化场景：1个物理CPU线程 = 1个vCPU 桌面云场景：1个物理核 = 4个虚拟机

说明：上述场景的计算方法仅供快速估算，准确vCPU个数、虚拟机密度与CPU型号、频率、domain0规格、虚拟机规格强相关，需要进行详细分析和计算。

在非欧几何中，平行线能相交，怎么解释？

发表于 2020-02-16 更新于 2020-02-17 分类于 文史数理

直线的本质就是两点距离最短，平面上就是我们通常理解的直线（线段）；球面上就是对应圆心的那个圆周。 因为你理解的平行线不是非欧几何中的概念，欧几里得几何是平面几何，所说的平行线是相对平面而言的，而非欧几何中平行线是相对曲面而言的。你所问的应该是：为什么黎曼几何中“过直线外一点，一条平行线也引不出来”。这得拿地球仪做参考，因为球面是弯曲的，球面上两点之间的最短距离是大圆周——用球面上的两点和球心三点做一个平面，截出的那个圆周就是大圆周（也叫短程线）。黎氏几何中的短程线就类似于欧氏几何（平面几何）中的直线。其实就是说，在一个大圆周之外，你不能再作一个大圆周跟它不相交。比如赤道是个大圆周，我们根本不可能再作一个大圆周跟赤道不相交。值得注意的是，地球仪上的纬线并不是大圆周，不是黎曼几何中的所说的“直线”（过两点之间一定是最短的那条线才是直线，这点不懂的话就别谈几何了），而在一个球面上，通过两点之间的最短距离是大圆周。黎曼几何不仅是爱因斯坦广义相对论的基础，而且广泛应用在航海和航空。

辩证法

发表于 2020-02-16 更新于 2020-02-17 分类于 文史数理

辩证法dialectic这个词在希腊文中的原意是“对话”“论辩”，本意就是对话。辩证法是希腊城邦民主制的产物，法国研究希腊的著名的学者让-皮埃尔·韦尔南有一本小册子叫《希腊思想的起源》，里面说哲学是城邦的女儿，哲学以城邦为基础产生起来的。首先的他讲城邦制度意味着话语具有压倒其他一切权利手段的特殊优势，以雅典为例，所有雅典公民是平等的，在政治生活之中，不是在靠官职地位财富，而是要靠语言来相互影响。要参加政治生活，就要使用语言，讲论辩，所以在希腊修辞学和论辩术就发展起来。这个所谓辩证法就是通过对话的方式来论辩，其实最初就类似一种思维游戏和思维训练，但是这样一种对话的方式，后来朝着两个不同的方向演变。一条是智者的诡辩，一条苏格拉底的助产术。

公元前五世纪的时期是智者的兴盛时期，但是智者的兴盛，走向诡辩，恰恰也是城邦民主制衰落的象征。因为智者不考虑真假对错是非曲直，他着重点就在论辩上，目的就是在论辩上取胜而不关注讨论话题的真假对错。被称为第一个智者的普罗泰戈拉有一句名言叫做“人是万物的尺度”，我们说人是万物的尺度，带有某种人类中心论等等。其实希腊那个时期跟我们现在理解会有距离，他所说的人不是大写的人不是抽象的人而是个人。所以当普罗泰戈拉说人是万物的尺度的时候，他的意思是说我们每个人都是尺度。换句话说，自然万物本没有尺度，你的尺度就是尺度，我的尺度也是尺度。我们每个人意见不一样那就尺度都看你自己。为什么如此，他还有一句话叫做一切理论都有其对立的说法。正面可以说，反面也可以说。恰恰就是这个，辩证法走向诡辩。

与此不同的一条道路是苏格拉底之路，苏格拉底在他那个时代往往也被人们误认为是智者，但是实际上苏格拉底用这种对话的方式，不是为了争强好胜，而是为了追求真理，要问出真假对错。苏格拉底哲学的问题落实在是什么的问题，问一个事物是什么，就是一个事物的本质规律。苏格拉底助产术的方法其实就是通过问答的方式，揭露对方的缺陷，一步一步把讨论的问题，从具体的个别的推向普遍的方式，最终追求能够把握真理。

说到苏格拉底，当然也会提到他的学生柏拉图。苏格拉底本人是述而不作，他的思想其实主要是由他的学生柏拉图记录下来，柏拉图流传于世的著作几乎绝大部分都是以他的老师苏格拉底为主，因此人们考证辩证法的概念是柏拉图最早提出来，苏格拉底是助产术，柏拉图的辩证法是一套非常复杂的辩证推理的方法，也可以叫做问答法，就是通过问和答来推进对对事物的认识。在这个问答法里边的有一个主角，这就是追问提问题的人。这个提问题的人通常也可以叫做辩证法家，他负责问，对方就负责回答。这问自然有技巧，其实、、在柏拉图的这种方式里边基本上是在这个辩证法家的控制之下，目的是追问真理，他有几篇和普罗泰戈拉，高尔吉亚这样的智者进行对话，进行正面交锋去讨论很多具体问题，也就说把这个以求知为目的的问答法就是辩证法与智者诡辩区别开。

但是把辩证法当做一种方法来论述的是亚里士多德。他是柏拉图的学生，其实这三代都在反对批评智者。亚里士多德里的辩证法也是一种推理，论证的逻辑，但是它和形式逻辑有区别，形式逻辑是要找到大前提，然后大前提小前提到结论三段论的推理。辩证法则是从经验出发，从不确定的东西出发，从假设出发沿着逻辑的道路，最终推动推论出确定知识的方法。所以辩证法和逻辑其实是有千丝万缕的联系，到了中世纪经院哲学，辩证法就是逻辑。中世纪早期教父哲学主要是新柏拉图主义的影响，十一世纪到十四世纪亚里士多德有很大影响。
