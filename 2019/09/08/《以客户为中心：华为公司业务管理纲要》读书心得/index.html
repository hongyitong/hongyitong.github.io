《以客户为中心：华为公司业务管理纲要》读书心得

发表于 2019-09-08 分类于 读书心得

第一篇 以客户为中心

为客户服务是华为存在的唯一理由

  • 未来的竞争是管理的竞争
  • 问题不可怕，关键是我们面对问题的态度，我们必须要有正确面对问题的态度，必须找到解决问题的正确方法；
  • 从泥坑中爬出来的人就是圣人。
  • 市场部在全世界刨那么多坑是好事，我们得赶紧去种树。
  • 华为公司的特征就是服务文化；下游就是上游的客户；以客户为中心就是帮助客户成功；
  • 通过序列流程化的组织架构和表格化的操作规范来保证满足客户需求。

华为的价值主张

  • 所谓商业模式，实际就是实现价值主张并获得相应经济回报的方式
  • 以客户为中心、以奋斗者为本、长期艰苦奋斗，是公司的核心价值观。坚持自我批判，是自我纠偏的机制。
  • 千古兴亡多少事，不尽长江滚滚来；
  • 而创新也是奋斗，是思想上的艰苦奋斗
  • 什么叫奋斗？为客户创造价值的任何微小活动，以及在劳动的准备过程中，为充实提高自己而做的努力，均叫奋斗，否则，再苦再累也不叫奋斗。
  • 胜则举杯相庆，败则拼死相救；
  • 世界上对男人最佳的表达是什么？电影《泰坦尼克号》告诉我们，在生死存亡的时候让女人先走，自己死掉，这就是对男人的最佳表达；
  • 运营商BG“三朵云”（体验云、知识云和客户方案云）
  • 一定要在战争中学会战争，一定要在游泳中学会游泳

质量是华为的生命

  • 服务就像黄继光在前面堵枪眼
  • 品牌就是承诺，品牌的根本是诚信；

深淘滩，低作堰

客户满意度是衡量一切工作的准绳

  • 成就客户就是成就自己
  • 客户的利益就是我们的利益

增长

  • 持续有效增长，当期看财务指标，中期看财务指标背后的能力提升，长期看格局以及商业生态环境的健康，产业的的可持续发展等。

追求长期有效增长

  • 企业的各种难题，只有在发展中才能得到解决。
  • 短期导向，会导致杀鸡取卵的取向；
  • 增长和利润的取舍，在一定的利润基础上，优先考虑增长。
  • 马太效应，强者恒强；
  • 没有战略思维是不行的，所以要盯到我们的增长上，盯到我们创造的总效益上，然后再考核人均效益； （人效）

产品发展的路标是客户需求导向

  • 从技术导向转变为客户需求导向
  • 客户骂你的时候就是客户痛得最厉害的地方，客户的困难就是需求；
  • 市场营销定位为“两只耳朵，一双眼睛”，一只耳朵倾听客户需求，一只耳朵听行业、技术发展趋势；一双眼睛紧盯竞争对手；
  • 要向客户提供针对性的综合解决方案，而不是单个产品。

创新是华为发展的不竭动力

  • 创新虽然有风险，但不创新才是最大的风险；
  • 什么事解决方案？解决方案不是以技术为中心，是以需求为中心，这是前端的；后端以技术为中心，是储备性的。
  • 解决方案应该面向客户的需求和应用场景；
  • 领先半步是先进，领先三步成先烈；
  • 心胸有多宽，天下就有多大；
  • 站在巨人肩膀上前进；
  • 歪瓜裂枣：枣是裂的最甜，瓜是歪的最甜

更多地强调机会对公司发展的驱动

  • 各个大公司的研发经费都在销售额的10%左右。
  • 对于有志者来说，永远都有机会。任何“时间晚了”的悲叹，都是无为者的自我解嘲。
  • 目前最重要的不是成本高低的问题，而是能否抓住战略机会的问题。抓住了战略机会，花多少钱都是胜利；抓不住战略机会，不花钱也是死亡。
  • 务实与务虚相结合。我们企业强调务虚，什么叫务虚呢？就是要利用机会强制地牵引公司前进，寻找企业发展的方向，比如说产品的定位方向、人才管理的方向等各种方向，形成这种方向来牵引公司前进；
  • 在模糊的情况下必须多线作战，当市场明晰时立即将投资重心转到主线上去。

聚焦主航道，坚持“压强原则”

  • 战略竞争力量不要消耗在非战略机会点上；
  • 什么叫主航道？世界上每个东西都有正态分布，我们只做正态分布中间的那一段。

开放、竞争、合作，构建良好的商业生态环境

  • 将来的竞争已经从企业之间的竞争演变为产业链之间的竞争；
  • 商业生态链的建设，本质上是一个在供应商、合作者和客户之间如何分配利益的问题。
  • 华为有什么事，捅捅也好，小不震则大震。早些知道什么错了，总比病重了好。

业务管理的指导原则

  • 工作就是要找准方向，主要矛盾就是工作方向，而方向就是前进的目标，所以能否抓住主要矛盾，关键在于是否有明确的战略目标。
  • 乱中求治与治中求乱，抓主要矛盾和矛盾的主要方面。
  • 坚定不移的战略方向，灵活机动的战略战术；
  • 坚持自己的价值观
  • 谋定而后动
  • 纲举目张
  • 不断激活组织，始终保持它的活力，不使它退化和沉淀；
  • 坚持以业务为主导，会计做监督的宏观管理方法和体系建设；

第三篇 效率

未来的竞争是管理的竞争

  • 什么东西都是可以买来的，唯有管理是买不来的；
  • 各级领导干部要从以前的埋头拉车转变到抬头拉车，不要只见树木，不见森林，更多顾及各级管理体系的建设；
  • 企业管理目标就是流程化的组织建设；
  • 向管理要效益
  • 价值->目标->策略->关键成果->执行计划
  • 过低的人均效益要求必然带来组织的过度膨胀和机构臃肿；
  • 提高效率，不是增加劳动强度，而是减少无效工作。减少重复劳动；

企业管理的目标是流程化组织建设

  • 华为组织改革的方向是有功能型的组织结构，转化为流程型的组织结构，并由IT支持这个组织的运作。基于流程来分配权力、资源和责任的组织，就是流程化组织；
  • 让听得见炮声的人来决策；
  • 让听得见炮声的人来呼唤炮火；
  • 以销售额为中心，是BG（业务集群）的责任；以利润为中心，是区域的责任；
  • 公司未来的建制，前端是对付不确定的精兵组织，后端是对付确定性的平台和共享组织。
  • 总部从管控中心向支持、服务、监控中心转变；
  • 后方部门对于前方部门可以要求功能对齐，但是不要求组织对齐；
  • 四算：概算、预算、核算、决算
  • 我们的改革要从以功能为中心转向以项目为中心；
  • 管理要标准化、简单化；不能过度监控。
  • 现在公司有些运作环节的信息流还跑不过物流；
  • 瓶颈管理
  • 高薪不能养廉，要靠制度养廉；

从客户中来，到客户中去，以最简单、最有效的方式实现流程贯通

  • 流程是一个团队做事的基本规则。成功的经验总结出来就是流程。什么事流程化建设？就是标准化、程序化、模板化。清晰的流程、重复运行的流程和工作一定要模板化。要把可以规范化的管理都变成扳铁路道岔，使岗位操作标准化、制度化；
  • 流程要先解决“通”的问题，先打通，再优化。流程通最根本的是数据要通。
  • 管理就是例外管理，但是必须要把例外管理转成理性管理，不断减少例外管理的数量；
  • 规范化管理让大家不舒服，但是还是要走规范化管理的道路。
  • 产品越来越时装化；变化很快
  • 要一次把事情做好
  • 抬头看目标、低头思责任
  • 要实现从刚性预算到弹性预算的转变。业务上去了，预算也相应弹上去，同时要提升机关的服务效率，在标准作业时间内完成审批，保证前线业务增长后就自然、合理、迅速地获得更多资源。

打造数字化全连接企业

  • 能买来的系统就买，不要自己做。
  • 数据是公司的核心资产，流程通最根本是数据要通；
  • 要求经营性财务数据要保存15年；
  • 信息安全关系到公司的生死存亡；
    但也不要防卫过度，例如不需要每个人都要配备一部防弹车；
  • 信息安全是公司重大的系统工程，要有架构性思考，从整体看如何构建未来安全环境；
  • 安全防护系统要敢于投入，采用业界最优秀的产品和技术。

管理变革的方针

  • 成功不是走向未来的可靠向导，危机意识才是。
  • 引进世界领先企业的先进管理体系，要“先僵化，后优化，再固化”
  • 中国人的劣根性之一就是永远不愿规范；
  • 要求改良而不是大的改革；
  • 存在就是相对合理，不要盲目修改，思考清楚后再动。
  • 英国的光荣革命，实现了君主立宪，比法国的大革命更有合理性；
  • 价值的产生不完全在于成本降低，提高竞争力和盈利能力才是最主要的目标。
  • 华为公司最大的浪费是经验的浪费。不断地总结经验，有所发现，有所创造，有所前进。
  • 善于总结
    案例有两种，一种是故事化的案例；一种是表格化的案例，可以帮助学员更好地掌握科学的方法，直接用在实际的工作里。
  • 每个人每个季度都要写案例，不写案例就写心得，多个心得叠加起来，就能写出案例来了。项目做完不输出案例就等于浪费。

文史数理

  • 阿房宫赋
    呜呼！灭六国者六国也，非秦也；族秦者秦也，非天下也。嗟夫！使六国各爱其人，则足以拒秦；使秦复爱六国之人，则递三世可至万世而为君，谁得而族灭也？秦人不暇自哀，而后人哀之；后人哀之而不鉴之，亦使后人而复哀后人也。

  • 什么是红蓝对抗
    在军事领域，演习是专指军队进行大规模的实兵演习，演习中通常分为红军、蓝军，演习多以红军守、蓝军进攻为主。类似于军事领域的红蓝军对抗，网络安全中，红蓝军对抗则是一方扮演黑客（蓝军），一方扮演防御者（红军）。在国外的话，进行渗透攻击的团队经常称做红队，在国内称为蓝队实际上应该是比较准确的叫法。

  • 六尺巷
    六尺巷传说是安徽桐城的地方民间传说故事。这是一则发生在清代康熙年间，桐城境内的一桩脍炙人口的民间故事。大学士张英的府邸与吴姓相邻。吴姓盖房欲占张家隙地，双方发生纠纷，告到县衙。因两家都是高官望族，县官欲偏袒相府，但又难以定夺，连称凭相爷作主。相府家人遂驰书京都，张英阅罢，立即批诗寄回，诗曰：
    千里修书只为墙，让他三尺又何妨。
    万里长城今犹在，不见当年秦始皇。
    家人得诗，旋即拆让三尺，吴姓深为感动，也连让出三尺。于是，便形成了一条六尺宽的巷道。
    张英（1637—1708年），字敦复，又字梦敦，号乐圃，又号倦圃翁，安徽桐城县人。清朝大臣，名相张廷玉之父。

  • 什么是蓝血十杰
    　　蓝血十杰是二次大战结束后，来自美国战时陆军航空队“统计管制处”的十位精英，被刚刚从老亨利·福特手中接过福特汽车公司控制权的亨利二世招致麾下，进入公司计划、财务、事业部、质量等关键业务和管理控制部门。
    　　　　从此，他们掀起了一场以数据分析、市场导向，以及强调效率和管理控制为特征的管理变革，使得福特公司摆脱了老福特经验管理的禁锢，从低迷不振中重整旗鼓，扭亏为盈，再现当年的辉煌。这十位精英所抱持的对数字和事实的始终不渝的信仰，以及对效率和控制的崇拜，使之获得了“蓝血十杰”的称号，人们将他们尊称为美国现代企业管理的奠基者。

  • 酬乐天杨州初逢席上见赠
    刘禹锡

             巴山楚水凄凉地， 二十三年弃置身。  
         怀旧空吟闻笛赋， 到乡翻似烂柯人。  
         沉舟侧畔千帆过， 病树前头万木春。  
         今日听君歌一曲， 暂凭杯酒长精神。

    “沉舟侧畔千帆过，病树前头万木春”！是啊，几只船虽然沉没了，但在它的旁边有无数只船顺风满帆，驶向远方。几棵树生病了或腐朽了，越过它们，却是万木争春，无限生机。这一名句寓意深刻，饱和哲理，被后人广泛传诵和引用。不论是漫漫人生路，还是人与人、国与国之间的纷纭关系，不可能一帆风顺，总会遇到各种各样的羁绊和坎坷。但是，一切的困难、挫折、不如意、不愉快都是暂时的、局部的，终将化为尘埃，烟消云散。继之而来的将是阳光灿烂，云白天蓝，绿水青山，百花争妍。满载的航船将顺风快帆，驶向理想的彼岸。
