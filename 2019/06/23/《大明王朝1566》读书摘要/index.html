《大明王朝1566》读书摘要

发表于 2019-06-23 分类于 读书心得（文学）

文章摘要

  • 授权柄于宦官，以家奴治天下；
  • 做官要三思：思危、思退、思变；
  • 孔曰成仁、孟曰取义
  • 官做到这个位置，“静气”二字已是必然的功夫；
  • 做事时不问可不可能，但问应不应该；
  • 民可使由之，不可使知之；
  • 共济时艰
  • 道德经云“天地不仁，以万物为刍狗；圣人不仁，以百姓为刍狗”。；
  • 苏东坡：古人不见今时月、今月曾经照古人；
  • 太史公：人固有一死，或重于泰山，或轻于鸿毛；
  • 爱吵架的从来就怕两种人，一种是任你暴跳如雷，他却心静如水；一种是挑你一枪，扬长而去。
  • 枕戈待旦：并非拿着枪睁眼坐待天明，而是心如空城，枕着一杆枪也安然睡了。
  • 在官场，礼节就是内容；
  • 铁打的营盘流水的兵；
  • 用对人是干大事第一要义；
  • 不迁怒，不贰过；
  • 衣不如新人不如故
  • 不痴不聋不做当家翁
  • 治大国如烹小鲜
  • 无心为过，虽过不罚；
  • 有子万事足，无官一身轻；
  • 国之干城
  • 行到水穷处，坐看云起时；
  • 该出手时就出手，得饶人处且饶人；
  • 不能谋万世者不能某一时；不能谋全局者不能谋一隅
  • 这个世上，真靠得住的就两种人，一种是笨人，一种是直人；
  • 天下无不是父母
  • 商鞅立木之法
  • 巧言令色，大奸似忠；
  • 圣人无恒心，以百姓之心为心；
