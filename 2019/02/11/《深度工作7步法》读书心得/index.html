    《深度工作7步法》读书心得

    发表于 2019-02-11 分类于 读书心得（社会科学）

    第1步 准备工作：合理管理精力和时间

    全景思维
    留白有助创新

    第2步 已经开始，如何坚持：把重点从结果转到过程

    远大目标、勇于开始、关注过程

    第3步 坚持的过程中，如何把握方向：提出指引你前进的问题

    灯塔问题：不断提醒自己，什么是对你最重要的那个问题。

    第4步 深入过程中，如何规避风险：做好收益规划

    投资组合、股权

    第5步 初具规模，如何做大：分配角色，设定进度

    项目经理，负责协调、管理，这样会让其他人有时间专注于工作本身。 （抬头看路、低头拉车； 分工有效提高工作效率的一个体现）
    好的经理人的特质：
    1、构想愿景和设定目标
    2、重视个人
    3、为团队工作创造抱持性环境

    柳传志著名的管理三要素：搭班子、定战略、带队伍

    第6步 规模扩大，构建商业模式：成为企业艺术家

    产品+商业模式 类似 写信+信封
    让你施展商业模式设计才华的另一项练习是混搭商业模式，看会发生什么。选取某领域的一种商业模式，把它运用到另外一个行业中，那将是怎样的情况呢？
    跨界！

    第7步 反思：培养跨学科思维的超通才

    商业和艺术的融合
    将学生改造成能以领导者的思维方式思考的领导者，而不是独立思考的技术专家。
    专注于抽象思维

    文章摘要

    • 商学院教的全是框架。
    • 雇主权：个人在工作中的发明，知识产权属于雇主。
    • 最小可行产品（Minimum viable product） 简称MVP
    • 倘若确实付出了真诚的努力，且不遗余力，那么，失败看上去就跟成功一样令人兴奋。
    • 将密码分开（非对称性加密 RSA），是自文艺复兴以来最具有革命性的加密概念。
      它解决了，置身于一个到处是不可信之人的世界，你如何与一个值得信任的人做交易。
    • 马基雅维利：只要目的正当，可以不择手段。 （守正出奇）
    • 管理是一门艺术，而非科学。我们要慎防那些试图把人类行为数字化和量化的人...
    • 固定成本、变动成本：固定成本指不管你的产量大小都会产生的费用，而变动成本指原料等的费用，只有生产时，它才会产生。
    • 从A点到B点
    • 学习心态：失败是学习的一个过程
    • 两种方式成长：提高效率（ 1 -> N ）、创新（0 -> 1)
    • 对待新事物，就像新生儿，允许他们犯错，给时间给他们成长。 任何新事物，要抱着让子弹飞一会的态度去接纳他们。
    • 拥抱不确定性
