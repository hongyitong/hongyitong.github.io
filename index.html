数学与人生

发表于 2025-12-10 分类于 随笔

花了一个多礼拜，《机器学习的数学基础》的笔记整理，终于大功告成。相比文学类读书心得，这类工作更耗时——不同工具对数学公式的渲染各有细微差异，整理过程中需要大量人工核对，颇为繁琐。

不过，在整理的同时，相当于把相关知识又复习了一遍。当再次翻看这些笔记时，突然理解了大学高代教授说过的一句话：“看高数，是一种消遣。”面对那些优雅的公式和严密的逻辑，人会自然地心无旁骛（wù），沉浸在数学那近乎完美的世界里。在那一瞬间，似乎能暂时远离现实中的纷扰与不确定，获得片刻宁静。

说到底，人生本身比数学更难理解。读完数学书，往往有醍醐灌顶的获得感；而读完关于人生的书（尤其文艺类），却只会加深对世界的无力感，让人对人生更加迷茫。

读书笔记之一：导言

发表于 2025-12-05 分类于 读书心得（自然科学）

英文原版《Mathematics for Machine Learning》，在过去大半年的时间里断断续续地坚持读完了。只能说，数学书永远是最耗费时间和脑力的。如今终于抽出空来，把阅读中的笔记整理出来，并按知识领域分成七篇文章。

数学是这个世界上最精确的语言！

在学习世界里，人们常说马斯克的“第一性原理”重要，又常用“概念清晰”来评价一个好学生。归根到底，这些都指向同一件事：抓住根本概念的重要性。这也正是《Mathematics for Machine Learning》最突出的特点。

当然，由于本书涵盖面广——线性代数、解析几何、微积分、概率等一网打尽——读者仍需要一定的数学基础。有些章节在推导上略显紧凑，必要时配合 ChatGPT 等工具理解概念，会大大提升效率。

这本书最大的价值在于：它并非简单罗列公式，而是从数学的角度、用数学的语言，把机器学习所需的数学概念系统梳理了一遍，并把各个知识点之间的关联讲得十分透彻。阅读过程中，你能明显感受到作者试图搭建的是一张“概念图谱”，而不是一本“知识清单”。

论孤独

发表于 2025-12-01 分类于 随笔

一直以来，都坚信读书和跑步的本质是孤独的，自己也常以“享受孤独”而自居。

随着阅读的积累，心得的增多，思考的拓展，阅历的增长，这个世界的面纱一层层的剥落，露出了其可怕的真相和悲剧的色彩（《悲剧的诞生》），孤独感也愈发强烈。

尼采说过，除了神灵、野兽和哲学家，人都忍受不了孤独（《格局》）。显然，我不属于前两者，离哲学家又甚远，离普通人又太近，因此，按尼采的说法，也不清楚自己究竟能忍受（享受）多久。

说到哲学家，尼采说过，需要兼备天才和疯子的要素，而我显然都没有。不过，尼采在《悲剧的诞生》中也提到，“就算人生是幕悲剧，我们也要有声有色地演这幕悲剧，不要失掉了悲剧的壮丽和快慰。”

以强大的生命力对抗人生悲剧，赋予人生以意义（《悲剧的诞生》）。这或许就是我们作为“第四种人”的人生路。

关于写读书心得

发表于 2025-11-30 分类于 随笔

如今，网络上各种读书心得已是汗牛充栋，有时候我也会犹豫、动摇，在这样的时代，坚持写读书心得还有意义吗？AI时代来临之后，这种举棋不定的想法尤为强烈，特别对我这种非文字工作者的理科生来说，写作本来就是一件苦差事。

然而，就像鲁迅说过，“一部红楼梦，经学家看见《易》，道学家看见淫，才子看见缠绵，革命家看见排满，流言家看见宫闱秘事。”正是这种关于阅读的个体性、独特性，给了我持笔不停的力量。

我，一个平凡得不能再平凡的人，没有各种“家”的深刻见解，却能在书里读到自己生活与影子。这份体验独一无二，是以斗胆记之，谓之以“心得”。这些心得定格了当时当刻的心境，捕捉了那时那地的自我。它有别于别的读者，更有别于AI的概率。

此外，还有另外一股力量在支撑着我，那就是，这片小小天地其实是自己内心的港湾。

在这纷纷扰扰的世界里，这片小天地，安静而平和，仿佛“苟全性命于乱世，不求闻达于诸侯”的小室。可以在这里轻声细语的诉说，可以在这里敞开心扉的记录，可以在这里无所畏惧的幻想，这里就是自己知音，它懂得何时是高山、何时是流水。

常常在想，如果有一天，不必为五斗米而折腰，选择自己喜欢的事情，那我的选择会是什么？

或许，就是读很多很多的书，写很多很多的心得，这也是我和这个世界最好的对话方式，也是我这个短暂的生命能留给这个世界最好的礼物，也是我作为一个生命过客的最大意义吧。

也希望真的能有这么一天。

文章精彩语句摘录

发表于 2025-11-26 分类于 随笔

刚开始写读书心得时，还会很认真地把书中的精彩语句逐条摘录出来，整理成文字附在文章末尾。后来嫌麻烦，就改成直接拍照，以图片形式放在文末。这样做确实更省事，还能把书页边栏里随手写下的批注原汁原味地保留下来。

不过，这种方式也有明显的缺点。 其一，阅读体验不佳。文字摘录可以做到一目十行，也方便后期搜索，而图片则不够友好； 其二，页面加载速度慢，尤其是多图时，打开文章会变得非常卡顿。

接下来计划逐步把这些图片重新整理为文字。现在的 AI 识别已经很成熟，可以快速完成图片转文字的工作，整体工作量也在可控范围之内。

关于喜欢

发表于 2025-11-22 分类于 随笔

年轻时，喜欢一个人就像麦当劳的广告词那样：我就喜欢；那时，喜欢一个人需要理由吗？不需要。需要吗？不需要。那时的喜欢就像小孩看到阳光就要奔跑一样，直接而单纯。

成年后，喜欢一个人就像林忆莲的那首歌“我就是喜欢你现在的样子”里面唱的，属于你的一切都是美丽。成年的喜欢是心动之上的欣赏和包容，欣赏他/她身上闪耀的光芒，接纳他/她身上的不足。懂得即使不完美，却仍愿意把温柔的目光停在对方身上。

喜欢的旅程，就是从“我喜欢这个感觉”，走向“我喜欢完整的你”。

关于跑团和读书会

发表于 2025-11-19 分类于 随笔

刚开始跑步、读书的时候，对参加所谓的跑团或读书会很感兴趣。但后来渐渐淡了，因为慢慢明白：跑步和读书本就是属于孤独的个人体验。跑步时的思绪涌动，是与自己的身体、与天地万物对话的过程；而读书则是与心灵、与灵魂交谈。俗话说，一千个读者就有一千个哈姆雷特，读书的过程，是将个人的生活体验与书本发生化学反应的过程；离开了自己的经历，别人所谓的“感同身受”往往是不存在的。所以，许多读书分享会反而更像是一种社交，而这与读书所强调的个体性的、心灵与书本交融的孤独本质，未免有些背离。跑步亦然。

正因如此，这十多年来，始终在这样的孤独中坚持着跑步，也坚持着读书。

那句话说得好：孤独是一个人的狂欢，狂欢是一群人的孤独。对我而言，跑步和读书，就是属于在自己小宇宙里的狂欢。
