智能体技能（Agent Skills）：关于扩展大语言模型功能的 Claude 技能的数据驱动分析

Agent Skills: A Data-Driven Analysis of Claude Skills for Extending Large Language Model Functionality

摘要

Abstract

智能体技能（Agent skills）通过可复用的、类似于程序的模块来扩展大语言模型（LLM）智能体，这些模块定义了触发条件、过程逻辑和工具交互。随着这些技能在公开市场上的激增，目前尚不清楚有哪些可用的技能类型、用户如何采纳它们以及它们会带来哪些风险。为了回答这些问题，我们对来自一个主要市场的 40,285 个公开上市的技能进行了大规模的数据驱动分析。

我们的研究结果表明，技能的发布往往在短期内爆发式涌现，并与社区关注度的转移相吻合。我们还发现，技能内容高度集中在软件工程工作流中，而信息检索和内容创作在用户采纳中占了很大份额。除了内容趋势外，我们还揭示了各类别之间显著的供需失衡，并且表明尽管长度呈现重尾分布，但大多数技能仍保持在典型的提示词预算（Prompt budgets）之内。最后，我们观察到强大的生态系统同质化，存在广泛的意图级冗余，并识别出了不容忽视的安全风险，包括允许改变状态或系统级操作的技能。

总的来说，我们的研究结果为智能体技能作为一种新兴的智能体基础设施层提供了量化快照，并为未来在技能复用、标准化和安全感知设计方面的研究提供了信息。

The New SDLC With Vibe Coding （Day 1）

发表于 2026-06-17 分类于 书籍翻译

https://www.kaggle.com/competitions/5-day-ai-agents-intensive-vibecoding-course-with-google/discussion?sort=hotness

Kaggle 《5-Day AI Agents: Intensive Vibe Coding Course With Google》 Day 1 白皮书

The New SDLC With Vibe Coding

From ad-hoc prompting to Agentic Engineering

全新的软件开发生命周期（SDLC） 伴随“氛围编码”（Vibe Coding）的崛起

从随意的主动提示词到智能体工程化（Agentic Engineering）

The most profound shift in software engineering isn't a new language, framework, or cloud service. It's the transition from writing code to expressing intent, and trusting intelligent systems to translate that intent into working software.

软件工程最深刻的转变，并不是出现了一种新的语言、框架或云服务。而是从“编写代码”向“表达意图”的跨越，以及去信任智能系统能够将这种意图转化为可以运行的软件。

引言

Introduction

在计算机历史的大部分时间里，编程一直是一种“翻译”行为：用人类的语言理解问题，用抽象的术语设计解决方案，然后用机器可以执行的语法将其呈现出来。每一个步骤都会引入摩擦。而现在，这种摩擦正在崩溃。软件工程正经历着自高级编程语言问世以来最重大的变革。几十年来，开发者与机器的主要交互界面一直是语法：花括号、分号、类型注解以及编程语言的精准文法。那个时代正在走向终结。

一个全新的范式已经到来，在这个范式中，开发者表达他们想要构建什么，而不是如何去构建它。实现过程由机器来处理。人类则负责提供意图、架构和审判与裁决。 这并非遥远的未来——而是对于快速增长的专业开发者群体而言，每天都在面对的现实。截至 2026 年初，已有 85% 的专业开发者定期使用 AI 编码智能体（AI Coding Agents），51% 的人每天都在使用，据估计，所有新代码中有 41% 是由 AI 生成的。

这种转变并非一蹴而就。它始于自动补全——编辑器中简单的 Token 预测。接着出现了行内代码建议，能够补全整个函数。随后，基于聊天（Chat）的交互界面允许开发者用自然语言描述功能，并接收可运行的实现方案。而现在，完全自主的智能体（Agents）已经能够克隆代码仓库、规划涉及多个文件的修改、在沙盒环境中执行这些修改、运行测试并提交拉取请求（Pull Request）——全过程甚至不需要人类输入单行代码。

DeepSeek-V3 技术报告

DeepSeek-V3 Technical Report

摘要

我们推出了 DeepSeek-V3，这是一个强大的混合专家（Mixture-of-Experts MoE）语言模型，其总参数量为 671B，每个 Token 激活的参数量为 37B。为了实现高效的推理和具有成本效益的训练，DeepSeek-V3 采用了在 DeepSeek-V2 中得到充分验证的多头潜在注意力（Multi-head Latent Attention MLA）和 DeepSeekMoE 架构。此外，DeepSeek-V3 开创了用于负载均衡的无辅助损失策略，并设置了多 Token 预测训练目标以获得更强的性能。我们在 14.8 万亿个多样化且高质量的 Token 上对 DeepSeek-V3 进行了预训练，随后进行了监督微调和强化学习阶段，以充分释放其能力。综合评估表明，DeepSeek-V3 的表现优于其他开源模型，并实现了与领先的闭源模型相当的性能。尽管性能优异，但 DeepSeek-V3 的完整训练仅需要 2.788M H800 GPU 小时。此外，其训练过程非常稳定。在整个训练过程中，我们没有遇到任何不可恢复的损失突增（loss spike），也没有进行任何回滚。模型权重可在 https://github.com/deepseek-ai/DeepSeek-V3 获取。

Dive into Claude Code: The Design Space of Today’s and Future AI Agent Systems

深入解析 Claude Code：当下与未来 AI Agent 系统的设计空间

Claude Code is an agentic coding tool that can run shell commands, edit files, and call external services on behalf of the user.

Claude Code 是一款智能体（agentic）代码编写工具，能够代表用户运行 Shell 命令、编辑文件并调用外部服务。

This study describes its comprehensive architecture by analyzing the publicly available TypeScript source code and further comparing it with OpenClaw, an independent open-source AI agent system that answers many of the same design questions from a different deployment context.

本研究通过分析其公开的 TypeScript 源代码，详尽地阐述了其整体架构，并进一步将其与 OpenClaw 进行了对比。OpenClaw 是一个独立的开源 AI Agent 系统，它在不同的部署语境下回答了许多相同的设计问题。

Our analysis identifies five human values, philosophies, and needs that motivate the architecture (human decision authority, safety and security, reliable execution, capability amplification, and contextual adaptability) and traces them through thirteen design principles to specific implementation choices.

我们的分析确立了驱动该架构的五个人类价值、哲学与需求（人类决策自主权、安全与保障、可靠执行、能力放大以及上下文适应性），并通过 13 项设计原则将其追溯至具体的实现选择。

《皮囊》读书心得

发表于 2026-06-02 分类于 读书心得（文学）

一位朋友推荐的一本书，或许前段时间阅读的是茅盾文学奖作品，相形之下，《皮囊》只能说中规中矩，有些许惊喜，但也不多。

作者蔡崇达和我差不多的年纪，我的先辈又是从福建迁居过来，所以书中作者描述小时候的生活习俗倒是非常相像，阅读过程也不断勾起了自己的童年往事，甚至有了写回忆录的冲动。

全书上半部，因为有时代背景代入感的原因，看得津津有味。后半部，总觉得作者是以一种成功者的优越视角在叙事，倒是有点不适。

用她的话说，那时候困难是普遍现象，因此困难显得很平常，显得不值一提。只是每个家庭要想办法去消化这种困难，并且最终呈现出波澜不惊的平凡和正常。

确实是，回想起小时候，改革开放前，窘迫的经济所带来的各种困难，相对来说，现在所面临的困难不可同日而语。我们那时还小，还懵懵懂懂，好奇大人们都是怎么过来的。不过可能也像作者所说吧，因为普遍就不值得一提了。
