人生哲思

发表于 2017-01-20 更新于 2024-03-14 分类于 生活杂记

本文是读书过程中的一些人生感悟。

2019.07.13

35、承受压力
有句话说的好，高管，公司请你来，有80%的工资是承担压力的价值；
所有工作之后，除了技术之外，也要学会承担压力。

34、守恒定律
守恒是宇宙的第一原则；
就像《断头女王》茨威格那句耳熟能详的名言想必大家都很熟悉。
“她那时候还太年轻，不知道所有命运赠送的礼物，早已在暗中标好了价格”。
日常中，包括物理学中的能量守恒，人生的命运守恒，人际交往中的价值守恒；都遵循着这个原则。

33、关于嘴笨
有些人尖牙利齿，有些人嘴笨；
俗话说，沉默是金，对于嘴笨的人，不要去争辩，有时候沉默是最有利的武器。

32、人的经历
人只有经历过磨练，在磨练中思考，才能得到成长；
俗话说，学而不思则罔，思而不学则殆，人生也一样，平时做事情就是学的过程，在遇到磨难会促进人的思考，通过思考才能促进人的成长；
就像学习，平时的作业是学的过程，总结分析是思的过程，只有做作业的基础上总结分析，才能达到螺旋式上升的目的。

31、关于职业规划
需要规划好自己的人生目标、职业目标；
然后趁年轻，如果在本单位没有有升职空间，可以考虑2-3年跳一次槽，优先考虑跳到大公司，通过跳槽来升职，逼近自己的职业目标。

2019.03.17

30、关于工作的三步曲
判断事情正否与否；敢于承担责任去做正确的事；相信自己的判断。

2018.07.21

29、关于职业生涯规划
开始尽量到大公司，能学习到很宏观、规范的流程和做法，同时，大平台在你将来的职业生涯会起到镀金的作用；
另外，如果在一个公司呆了2-3年，如果没有很好的发展空间，又不能学到很多东西，一定要换一个单位。

2018.06.11

大考，例如中考、高考前，最重要的鼓励孩子的一句话就是，让孩子想想自己考得最好的一次是怎么做的。
避免让孩子不要紧张之类的话，甚至可以和孩子说，紧张是一个人正常的反应，适度的紧张有利于发挥。
28、学会过滤噪音。
有些信息是噪音，所以你在接受到信息的时候，需要冷静分析，找出问题的本质，同时过滤掉噪音，不要受噪音的干扰。
27、成熟的定义，就是你会换位进行思考了。
26、自信的定义
自信就是明白自己想要什么，展现真实的自己，表达自己真实的想法；接受真实的自己，包括自己的长处和不足。
25、跑步的感悟：昨晚起风，天变冷，换跑步衣服的时候，很冷，都想打退堂鼓了，下去的时候，风吹着更厉害，思想斗争就更厉害，不过还来还是决心跑下去，后面就感觉很舒服了。就像人生改变的时刻一样，各种犹豫和举棋不定，其实，坚持下去一定能迎来不同的体验；
还有跑步的目标，当一开始，你定的是10公里，那你会跑到10公里才会觉得累，但是如果你定的目标是5公里，那你跑到5公里的时候就会感觉累了。人生也一样，你的目标就决定了你的动力。
24、所谓的人生心灵鸡汤都是鸦片。真正的人生就是逆水行舟，不进则退，只有勇敢的离开舒适区，才能让自己的人生、自己的家庭得到保护。
23、《捉妖记2》：一家人在一起，即使坐牢也是开心的；
《红海行动》：勇者无惧，强者无敌。

22、认识自己，确实是一个最重要的事情，也是人一辈子追寻的东西。认识自己的长处、短处，然后找到自己的位置、适合自己的工作和生活等。
21、管理就是需要先满足股东利益的前提下去满足员工的利益，而不能倒过来！
20、年轻时候喜欢一首歌，是因为旋律；长大后喜欢一首歌是因为歌里面有你的故事。
19、从整体的角度（全局）来考虑，从长期（未来）的角度来思考问题。
18、小孩的思维
早上地铁过安检的时候，一个大概3岁的小孩，死活都不肯把他的书包拿出来过安检机。
可能在小朋友的世界里，一旦脱离他的视线，就是消失了。所以，上幼儿园的时候，小朋友的分离焦虑症也是同一个道理，他们可能以为分开之后，爸爸妈妈就会消失了;)

17、关于自信
自信就是说出你心中真实的想法，需要的时候坚持它。
16、游戏人生
人们常说，人生就像一场游戏，来让人能豁达的应对人生的起伏和风雨。
所以说，在面对人生时，首先要学会如果面对游戏中的输赢 ；-）
15、关于改变
一般改变总是有原因的，不够痛就不会动，有的痛是切实感受到的，有的是对未来的焦虑。
14、乐观的人适合当领导（领路人），悲观的人适合当执行者。
因为乐观的人，对未来、对目标持乐观态度，能感染团队成员努力达成目标；悲观者因为害怕失败，所以在处理事情上会考虑周全。
13、习近平：“只要路走对了，就不怕遥远。”
12、生活总是让我们遍体鳞伤，但到后来，那些受伤的地方一定会变成我们最强壮的地方。——《老人与海》
11、胸有惊雷而面如平湖者，可拜上将军
顺，不妄喜；逆，不惶馁；安，不奢逸；危，不惊惧；胸有惊雷而面如平湖者，可拜上将军。
起初只是喜欢后边两句，以为大将之风至高境界，胸有惊雷，海纳百川，吞雷之势不溢于言表，神人也。
后来到处找此句出出，居然找出以下几个调调：
胜，不妄喜；败，不遑馁；胸有激雷而面如平湖者，可拜上将军！出自《孙子兵法》（三国之孔明有引用以教人）
原文是：顺，不妄喜；逆，不惶馁；安，不奢逸；危，不惊惧；胸有惊雷而面如平湖者，可拜上将军。
石奢者 楚昭王相也。等等皆有引用，但其根基皆出至于兵法卷中
胜利了，不妄（应该是这个妄）自狂喜，失败了，不惶恐气馁，胸有大志而不露声色的人，可成大事（可拜他为大将军）
10、关于抢时间
早上坐地铁的时候，有些人总是抢着上车、抢着做电梯，一眼看过去，大多数这些人都是偏底层一点。其实也反过来说明了一个问题，他们宁愿急急忙忙的上班，也不愿意早起10分钟，然后从从容容的上班去。他们的时间概念或者说时间管理是有问题的。
9、关于形式化
形式化其实就是中国自古以来的礼仪，形式化能产生一种仪式感。所以在日常中生活中，形式化是很有必要的，而客套话其实也是形式化的一部分。
8、不迁怒，不二过
在经典《论语》的《雍也第六》有这么一段文字，摘录如下：哀公问：“弟子孰为好学？”孔子对曰：“有颜回者好学，不迁怒，不二过。不幸短命死矣！今也则亡，未闻好学者也。”
7、访谈
给答话者以思考时间，不要去建议这样那样的答复或提出另外的问题；谈话过程中的暂停有利于答话人回忆一些最关键的信息。
要谦虚，答话人是专家，而不是你的访谈人。
6、关于事物的本质
我们经常说，看事情要看到本质，其实这个本质就是千变万化表面下的不变。古话说，万变不离其宗，这个宗就是不变的东西。例如，牛顿能从各种复杂的运动中，总结出F=ma，这就是事物运动之后不变的东西。例如，每个知识点的习题千变万化，但其不变的都是概念和定理。所以我们生活和学习必须学会抓住不变的东西入手。
同时，在发现本质之后，在描述本质之后，还需要给这些描述予以命名，这样才能方便传递、交流、理解。一个意简言赅的命名是非常关键的。
例如，以前我们都说，你看到的世界就是你内心的反映，这句话说到了本质，但是稍显繁琐。现在定义出“ABC理论”，A是引发你情绪的事件，B是你的信念或你对事物的诠释，C就是结果。其实引发C的不是A，而是B，而我们唯一能控制和改变的就是B，但我们通常会往往在A及相关事情上纠缠。
5、选择的问题
印度电影《为爱毁灭》中，盲女的父亲对盲女说：“我们的明天建立在今天的基础之上，我们今天选择做什么，将会决定明天发生在我们身上的事情。选择并不是在好与坏之间，那太容易了。生活真正的选择是在两个坏中选择不太坏的，或是两个好中选择更好的。”
其实这就是生活，并且在你选择的同时必须承担好中同时带给你的坏，这就是承担！
4、如何看待考试
考试回来，不要问对错在哪里，也不要和谁比较怎么怎么样，要问进步在哪里。
3、看书的优势
看书和看新闻的差别：看书，你看了一阵，会去思考后面会怎么样，有一股强烈看下去的愿望，并且那段时间会沉醉在书中有关联的深度的故事当中。看新闻，一会看完一条，马上要寻找另外一条来看，思维需要不断的切换，人就特别累，看到后面，签名也都忘了。
2、做事情的判断问题
做任何事情之前，先判断对错；
然后进行价值判断，是否值得。
例如有一次一个下属问我，通知业务周六切换文件服务器好不好，我就顺其自然的问了情况，然后提出了应该怎么做怎么做。后来运维对由项目组来提出这件事有意见。反思：接到事情，应该第一反思是不是应该做，再去考虑怎么做。
另外一件事，就是一个业务直接打电话给我问某个文件审批的问题，我在开会，就随口说了让他们发单号给我。反思：后来想想，这些事情就应该让他们先去找经办人处理，不该直接找到我。
1、兴趣的问题
数学、物理：儿子的数学一直在我的监督之下学习，而物理都是他自己在学习；但是儿子对物理的兴趣远远大于数学。分析了之后，其实数学因为一直在发现他的问题，所以导致他的兴趣不高，而物理我只是在鼓励他、聆听他的想法，所以反而他更有兴趣了。
