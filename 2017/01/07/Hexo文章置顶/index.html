Hexo文章置顶

发表于 2017-01-07 分类于 博客搭建

在hexo github 的issue里找到了解决办法,解决Hexo置顶问题，只需两步：

  1. 用文章中的js代码替换node_modules/hexo-generator-index/lib/generator.js (见下文代码段)
  2. 在需要置顶的文章的front-matter中添加top值，值越大越置顶。
